# Academy-Top-Manual-testing

#### Решения домашних заданий "Мануальное тестирование"

## Homework 01

### Тема: "Введение в тестирование. Ошибка, дефект, сбой. Роль QA"

**Задание 1. Разбор трех ситуаций**    
Выберите игру, приложение или сайт.    
Найдите 3 проблемные ситуации и для каждой заполните:
1. ) Контекст (платформа, версия, условия).
2. ) Предполагаемый error (где могла возникнуть человеческая ошибка).
3. ) Defect (что именно в системе работает неправильно).
4. ) Failure (что видит пользователь).
5. ) Риск для пользователя.
**Минимальный объем: 3 полноценных ситуации.**
